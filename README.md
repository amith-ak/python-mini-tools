# Python Mini Tools

This project is a collection of simple Python command-line utilities created to practice and demonstrate core programming concepts.

## Features
- Calculator (supports +, -, *, / operations)
- Random Password Generator
- Number Guessing Game

## Concepts Used
- Functions
- Loops (while and for)
- Conditional statements
- Exception handling
- Random module
- User input handling

## How to Run

1. Install Python (version 3.x)
2. Download the project
3. Open terminal or command prompt
4. Run the file:

python mini_tools.py

## Purpose of the Project
This project was created as part of my learning journey in Python programming to strengthen problem-solving and logic building skills.
